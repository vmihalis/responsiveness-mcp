  <name>Task 1: Create landing page with inline CSS and JavaScript</name>
  <files>site/index.html</files>
  <action>
Create site/index.html with the following structure:

1. **HTML Head** with:
   - Meta charset, viewport
   - Title: "screenie - Responsive Screenshots in One Command"
   - Meta description: "Capture responsive design screenshots across 57 device viewports with one command. No setup required."
   - Open Graph tags (og:title, og:description, og:image, og:url)
   - Twitter Card tags (summary_large_image)
   - Favicon link
   - ALL CSS inlined in a style tag (no external stylesheet)

2. **Inline CSS** implementing:
   - CSS custom properties for colors (supports dark mode)
   - System font stack: system-ui, -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, "Helvetica Neue", Arial, sans-serif
   - Monospace for code: ui-monospace, SFMono-Regular, "SF Mono", Menlo, Consolas, "Liberation Mono", monospace
   - Minimal reset (box-sizing, margin/padding zero)
   - Dark mode via @media (prefers-color-scheme: dark)
   - Mobile-responsive layout
   - Install box styling with prominent copy button
   - Hover states for interactive elements

3. **Page Body** containing:
   - Hero section with:
     - h1: "screenie"
     - Tagline p: "Capture responsive design screenshots across 57 device viewports with one command"
     - Install box with code element showing "npx screenie https://your-site.com" and Copy button
     - Demo GIF img with explicit width/height attributes, loading="eager"
     - Links nav with GitHub, npm (add docs link placeholder pointing to #)

4. **Inline JavaScript** (at end of body):
   - copyInstallCommand() function using navigator.clipboard.writeText()
   - Include fallback using document.execCommand for older browsers
   - Visual feedback: button text changes to "Copied!" for 2 seconds

Color scheme:
- Light: white background (#ffffff), dark text (#1a1a2e), accent indigo (#6366f1)
- Dark: dark background (#1a1a2e), light text (#f9fafb), lighter accent (#818cf8)

<task type="auto">
  <name>Task 2: Create favicon, OG image, and copy demo.gif to site/</name>
  <files>site/favicon.ico, site/og-image.png, site/demo.gif</files>
  <action>
1. **Copy demo.gif** from demo/ to site/:
   cp demo/demo.gif site/demo.gif

2. **Create favicon.ico**:
   Create a simple 32x32 favicon using ImageMagick (convert command):
   - Purple/indigo background (#6366f1)
   - White "S" letter centered
   - Command: convert -size 32x32 xc:"#6366f1" -fill white -font "DejaVu-Sans-Bold" -pointsize 24 -gravity center -draw "text 0,0 'S'" site/favicon.ico

   If ImageMagick not available, create a minimal valid .ico file or use a base64 inline favicon in the HTML as fallback.

3. **Create og-image.png** (1200x630 for social sharing):
   Create using ImageMagick:
   - Dark background (#1a1a2e)
   - "screenie" in large white text
   - Tagline below in smaller gray text
   - Command: convert -size 1200x630 xc:"#1a1a2e" -fill white -font "DejaVu-Sans-Bold" -pointsize 96 -gravity center -draw "text 0,-80 'screenie'" -fill "#9ca3af" -pointsize 36 -draw "text 0,80 'Responsive Screenshots in One Command'" site/og-image.png

   If ImageMagick fails, create a simple PNG with available tools or document the manual step.

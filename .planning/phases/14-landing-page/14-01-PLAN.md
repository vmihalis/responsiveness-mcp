Create landing/index.html with the following structure:

1. **HTML Head** with:
   - Meta charset UTF-8, viewport for mobile
   - Title: "screenie - Responsive Screenshots in One Command"
   - Meta description: "Capture responsive design screenshots across 57 device viewports with one command. No setup required."
   - Open Graph tags (og:title, og:description, og:image pointing to https://screenie.xyz/og-image.png, og:url)
   - Twitter Card tags (summary_large_image)
   - Preload hint for demo.gif: `<link rel="preload" as="image" href="demo.gif">`
   - ALL CSS inlined in a style tag (no external stylesheet)

2. **Inline CSS** implementing Catppuccin Mocha theme (from research):
   ```css
   :root {
     /* Catppuccin Mocha colors - matches demo GIF */
     --ctp-base: #1e1e2e;
     --ctp-mantle: #181825;
     --ctp-surface0: #313244;
     --ctp-text: #cdd6f4;
     --ctp-subtext0: #a6adc8;
     --ctp-blue: #89b4fa;
     --ctp-mauve: #cba6f7;
   }
   ```
   - System font stack: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto, sans-serif
   - Monospace: ui-monospace, "Cascadia Mono", Consolas, monospace
   - Minimal reset (box-sizing, margin/padding zero)
   - Centered hero layout with max-width: 900px
   - Install box with surface0 background, blue accent button
   - Hover states for button and links

3. **Page Body** with centered hero section:
   - h1: "screenie" (in mauve color --ctp-mauve)
   - Tagline p: "Capture responsive design screenshots across 57 device viewports with one command." (in subtext0)
   - Demo GIF: `<img src="demo.gif" alt="screenie capturing responsive screenshots" class="demo" width="850" height="450">` - NO lazy loading (above fold)
   - Install box div containing:
     - code element with id="cmd": "npx screenie https://your-site.com"
     - button with onclick="copyInstall()" and class="copy-btn": "Copy"
   - Links nav with:
     - GitHub: https://github.com/memehalis/screenie
     - npm: https://www.npmjs.com/package/screenie

4. **Inline JavaScript** (at end of body):
   ```javascript
   async function copyInstall() {
     const cmd = document.getElementById('cmd').textContent;
     try {
       await navigator.clipboard.writeText(cmd);
       const btn = document.querySelector('.copy-btn');
       btn.textContent = 'Copied!';
       setTimeout(() => btn.textContent = 'Copy', 2000);
     } catch (err) {
       console.error('Copy failed:', err);
     }
   }
   ```

Use the complete index.html structure from 14-RESEARCH.md as reference but update Open Graph URLs to https://screenie.xyz/og-image.png.

<task type="auto">
  <name>Task 2: Create assets and netlify.toml</name>
  <files>landing/demo.gif, landing/og-image.png, netlify.toml</files>
  <action>
1. **Create landing/ directory and copy demo.gif:**
   ```bash
   mkdir -p landing
   cp demo/demo.gif landing/demo.gif
   ```

2. **Create og-image.png (1200x630)** using ImageMagick:
   ```bash
   convert -size 1200x630 xc:"#1e1e2e" \
     -fill "#cba6f7" -font "DejaVu-Sans-Bold" -pointsize 120 -gravity center -draw "text 0,-60 'screenie'" \
     -fill "#a6adc8" -pointsize 36 -draw "text 0,80 'Responsive Screenshots in One Command'" \
     landing/og-image.png
   ```

   If ImageMagick unavailable, create a simple placeholder PNG or note for user.

3. **Create netlify.toml** in repository root:
   ```toml
   [build]
     publish = "landing"
     command = "echo 'Static site - no build needed'"

   # Redirect www to apex
   [[redirects]]
     from = "https://www.screenie.xyz/*"
     to = "https://screenie.xyz/:splat"
     status = 301
     force = true

   # Security headers
   [[headers]]
     for = "/*"
     [headers.values]
       X-Frame-Options = "DENY"
       X-Content-Type-Options = "nosniff"
       Referrer-Policy = "strict-origin-when-cross-origin"

   # Cache static assets for 1 year
   [[headers]]
     for = "/*.gif"
     [headers.values]
       Cache-Control = "public, max-age=31536000, immutable"

   [[headers]]
     for = "/*.png"
     [headers.values]
       Cache-Control = "public, max-age=31536000, immutable"
   ```
  </action>
